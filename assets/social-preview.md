# social-preview.png blueprint

Use this to create the repository social preview image in GitHub settings.

## Export specs

- File name: `social-preview.png`
- Size: `1280x640`
- Style: dark, minimal, high-contrast, tech/editor aesthetic

## Layout

1. Background: dark navy-to-black gradient with subtle grain.
2. Top-left small label: `orbital-skills`.
3. Main headline (center-left): `AI skills for builders who ship`.
4. Subheadline: `Curated by @sguestsett`.
5. Right side visual: clean orbital line motif + tiny code/prompt card.
6. Bottom row chips: `hard-work`, `AI workflows`, `open-source`.

## Color + type

- Background: `#060A12` to `#0F172A`
- Accent: `#5EEAD4` and `#60A5FA`
- Text: `#E5E7EB`, secondary `#94A3B8`
- Font: Inter / Sora / Geist (semi-bold headline)

## Ready-to-use image prompt

"Design a modern GitHub social preview image, 1280x640, dark gradient background, minimal futuristic aesthetic, subtle orbital rings on right side, left-aligned text: 'AI skills for builders who ship', smaller line 'Curated by @sguestsett', tiny chips at bottom saying 'hard-work', 'AI workflows', 'open-source', high contrast, crisp typography, no clutter, premium developer brand style."

## Optional tweet-launch variant text

- Headline: `Stop writing random prompts.`
- Subheadline: `Use reusable AI skills. Start with hard-work.`
